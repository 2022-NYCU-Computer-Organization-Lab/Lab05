## Instructions that needed to support
- And, Or, Add, Sub, Xor, Slt (R-type)
- Addi, Lw, Slti, Slli (I-Type)
- Nop (This is encoded as ADDI x0, x0, 0)
- Sw (S-Type)
- Beq (SB-Type)
- Jal (UJ-Type)
